                  OPEN PASSWORD GENERATOR

System requirements

Windows 2000 or higher for 32 system
Windows Vista or higher for 64 system

Description and usage

This program is automatic password generator.
Just set a password length and click on "Generate" button for generate password.

Limitation

Maximum password length is 255.
