System requirements

Windows 2000 or higher for 32-bit systems
Windows Vista or higher for 64-bit systems

Description and usage

This program is a simple password generator.
Just set a password length and click on the "Generate" button to generate the password.

Limitation

Maximum password length is 99 characters.
